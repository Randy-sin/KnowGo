# KnowGo 产品文档

## 📋 概述

KnowGo 是一个基于 Next.js 15 构建的个性化 AI 学习平台，采用硅谷风格的极简设计。应用引导用户完成无缝的三步学习旅程：搜索 → 配置 → 学习。

---

## 🗂 文件结构与页面映射

### **核心应用文件**

| 页面/组件 | 文件路径 | 路由 | 描述 |
|---|---|---|---|
| **主页** | `app/page.tsx` | `/` | 带搜索界面的主要着陆页 |
| **配置页** | `app/configure/page.tsx` | `/configure` | 学习偏好设置 |
| **学习页** | `app/learn/page.tsx` | `/learn` | 个性化内容展示 |
| **布局** | `app/layout.tsx` | - | 根布局与元数据 |

### **支持文件**

| 文件 | 路径 | 用途 |

## 🚀 页面详情

### **1. 主页 (`app/page.tsx`)**

**路由：** `/`

**用途：** 用户查询的入口点

**主要功能：**
- 极简搜索界面
- 快速建议按钮
- 直接跳转到配置页面
- 硅谷风格设计

**代码亮点：**
```typescript
- useState 用于输入管理
- localStorage 用于查询持久化
- 直接 router.push 到 /configure
- Framer Motion 动画
```

**用户流程：**
1. 用户输入学习查询
2. 点击提交或建议
3. 重定向到配置页面

---

### **2. 配置页 (`app/configure/page.tsx`)**

**路由：** `/configure`

**用途：** 个性化学习体验

**主要功能：**
- 知识水平选择（初学者/中级/专家）
- 学习风格选择（12种不同风格）
- 带返回按钮的极简导航
- 实时进度指示器

**代码亮点：**
```typescript
- 复杂的选择状态管理
- 高级 Framer Motion 动画
- 选中状态的条件样式
- 配置的本地存储持久化
```

**用户流程：**
1. 选择知识水平
2. 选择学习风格
3. 进入个性化学习内容

**可选学习风格：**
- 课堂式、讲故事、对话式、导师式
- 深度式、快速式、诗意式、轻松式
- 案例式、孔子式、叙述式、爱因斯坦式

---

### **3. 学习页 (`app/learn/page.tsx`)**

**路由：** `/learn`

**用途：** 展示个性化AI生成内容

**主要功能：**
- 多步骤加载动画
- 基于配置的个性化内容
- 清洁的内容展示
- 极简导航

**代码亮点：**
```typescript
- 逐步加载模拟
- 动态内容生成
- 基于配置的定制化
- 清洁的markdown风格内容展示
```

**加载步骤：**
1. "分析您的问题"
2. "适配您的水平"
3. "制作个性化内容"

## 🎨 设计系统

### **配色方案**
- **背景色：** `#FFFFFF`（纯白）
- **前景色：** `#1A1A1A`（深灰）
- **边框色：** `#E5E7EB`（微妙灰）
- **静音文字：** `#6B7280`（次要灰）

### **字体系统**
- **字体栈：** `ui-sans-serif, system-ui, -apple-system, 'Segoe UI'`
- **特大标题：** `2.5rem-5rem`（响应式）
- **大标题：** `1.5rem-2.25rem`（响应式）
- **大正文：** `1.125rem`

### **动画系统**
- **缓动函数：** `cubic-bezier(0.16, 1, 0.3, 1)`
- **持续时间：** `0.2s-0.6s`
- **原则：** 微妙、有目的、不分散注意力

## 💾 数据流

### **LocalStorage 使用**
- `knowgo-query`: 存储用户的学习问题
- `knowgo-config`: 存储选择的水平和风格偏好

### **状态管理**
- React useState 用于组件级状态
- LocalStorage 用于跨页面持久化
- 无外部状态管理库（保持极简）

## 🛠 技术栈

- **框架：** Next.js 15 (App Router)
- **语言：** TypeScript
- **样式：** Tailwind CSS + 自定义设计系统
- **动画：** Framer Motion
- **图标：** Lucide React
- **部署：** 可部署到 Vercel/Netlify

---

## 📱 响应式设计

- **移动优先：** 针对移动设备优化
- **断点：** Tailwind 的默认断点
- **自适应字体：** 基于 clamp 的响应式字体
- **触摸友好：** 合适的点击目标和间距

---

## 🎯 设计原则

1. **极简主义：** 移除不必要的元素
2. **清晰度：** 清晰的视觉层次
3. **功能性：** 功能胜过装饰
4. **一致性：** 跨页面统一体验
5. **性能：** 轻量且快速
6. **可访问性：** 高对比度和可读性
